<h1><strong>What is Helden?</strong></h1>
<p>There was a YouTuber project called Minecraft Helden. After the project ended, there was only a download for Minecraft Java, as the developers hadn't created an official add-on for Bedrock. For those unfamiliar with Helden: It's a project in which different players compete against each other. If you die during a fight, you lose a heart. If you log out during a fight, you also die and lose a heart. Anyone who loses all three hearts is banned from the project unless you have deactivated the player&nbsp;<span style="color:#f1c40f">banbydeath</span>. This setting ensures that after the third death, you are put into spectator mode and not banded. This setting is enabled for every player by default.</p>

<h1><strong>Slash Command</strong></h1>
<p><span style="color:#f1c40f">/helden</span></p>
<p>opens the settings</p>
<p>&nbsp;</p>
<p><span style="color:#f1c40f">/heard name 0 - 4</span></p>
<p>With the command you can adjust the current hearts</p>
<p>&nbsp;</p>
<p><span style="color:#f1c40f">/banbydeath true/false</span></p>
<p>Prevents a player from being banned after the 4th death</p>
<p>&nbsp;</p>
<p><span style="color:#f1c40f">/duel</span></p>
<p>checks how many duels you still have</p>
<p><span style="color:#f1c40f">/duel name</span></p>
<p>challenges the specified player to a duel</p>
<p><span style="color:#f1c40f">/duel name accept/deny</span></p>
<p>accepts or rejects a duel</p>
<p>&nbsp;</p>
<p><span style="color:#f1c40f">/linkheard name</span></p>
<p>checks who is linked</p>
<p><span style="color:#f1c40f">/linkheard name true/false</span></p>
<p>with false you prevent the person from being linked to another</p>
<p><span style="color:#f1c40f">/linkheard name1 name2</span></p>
<p>With the command the linkheads can be customized</p>
